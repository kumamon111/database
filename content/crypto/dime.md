### 技術詳細


Specifications:

- Based on Quark
- CPU mining
- 64 sec Block
- 65536 sec (~18hrs & 1024 blocks) difficulty change (max. 10% up/50% down)
- 80 confirms
- 1024 coins per block  (halving every 512000 blocks ~ 375 days )
- Total ~460 billion coins

Block reward:

Block 1 = 1024

Block 2 = 2048

Block 3 = 3072
...
Block 1024 = 1048576

Block 1025 = 1024

(Block number % 1024 * base coins)
